# 💰 Finance - Kişisel Gider Takip Uygulaması

React Native ve Expo ile geliştirilmiş, modern ve kullanıcı dostu bir gider takip uygulaması.

### ✨ Özellikler
* 📊 **Pasta Grafik:** Harcamalarınızı kategorilere göre görselleştirin.
* 📅 **Takvim Seçici:** Harcamalarınızı tarih seçerek kolayca ekleyin.
* 🌓 **Tema Desteği:** Tek tuşla Karanlık (Orijinal) ve Aydınlık mod arasında geçiş yapın.
* 🛡️ **Hata Kontrolü:** Boş veya hatalı girişleri engelleyen doğrulama sistemi.
* 🍔 **Kategoriler:** Yemek, Ulaşım, Market ve Diğer seçenekleriyle harcamalarınızı ayırın.

### 🚀 Nasıl Çalıştırılır?
1. Bu projeyi bilgisayarınıza indirin (Clone).
2. Terminalde `npm install` komutu ile kütüphaneleri yükleyin.
3. `npx expo start` ile uygulamayı başlatın.
